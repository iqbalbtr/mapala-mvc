Nama Proyek: UKM 
Teknologi: php native, css, html, javascript

Pola Proyek: MVC

Kebutuhan sistem :
1. php >= 8

Instalasi database:
1. Jalankan phpMyAdmin
2. Buat database bernama db_ukm
3. import file database menggunakan file database.sql yang berada dalam root Proyek


Menjalankan Proyek:
1. ekstarak file
2. Masuk ke direktori public || cd public
3. Pastikan php versi 8 keatas sudah terinstal di dalam komputer || php -v
4. Jalankan Proyek || php -S localhost:3000
